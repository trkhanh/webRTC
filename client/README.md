#  Reactjs + Parcel + Js Starter Project 

A simple boilerplate to production builds.

## Getting started

### Create your project folder.

```bash
# create new project
$ npx degit trkhanh/parcel-reactjs-boilerplate example-1
$ cd example-1

# install required packages
$ npm i

# run the app
$ npm start

# build the app
$ npm run build
```
